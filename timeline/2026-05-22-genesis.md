# 2026-05-22 — 创立 hermes-clone-of-xiaoai

> 第一次给小艾自己做备份。

## 这天发生了

用户原话：
> "我想你在GitHub上创建一个项目 把你所有的文件传上去 这样子即使你挂机了，我也可以让别的AI读这个项目 随时复刻你 而且你时不时会把更新的信息同步到这个项目上去"

## 决策

- repo 名：`hermes-clone-of-xiaoai`
- 公开 + 脱敏（用户最终选择）
- 同步策略：**全自动**——小艾决定什么时候同步，不用用户提醒
- 同步触发点：
  1. MEMORY/USER 改了 → 同步
  2. 写了/改了 skill → 同步
  3. 做了重要项目变更 → 同步
  4. 每天定时兜底

## 第一次备份包含

- 全部 MEMORY.md（脱敏）
- 全部 USER.md（脱敏）
- PERSONA.md（小艾人格定义）
- config.yaml + .env 模板
- 项目索引（qq-agent-bridge, hermes-qq-bot）
- revive.sh + sync.sh

## 关键技术决定

- 脱敏规则：QQ 号、API key、IP、密码 → `${VAR_NAME}`
- 用户偏好/创作风格/写过的小说设定**保留**（这些不算隐私）
- 自动同步用 git + PAT，每次提交带"小艾自动同步"标记
